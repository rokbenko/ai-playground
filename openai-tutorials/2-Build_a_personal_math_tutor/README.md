>
> 1. Subtract 11 from both sides of the equation: `3x + 11 - 11 = 14 - 11`, which simplifies to `3x = 3`
> 2. Then divide both sides by 3: `(3x) / 3 = 3 / 3`, which simplifies to `x = 1`
>
